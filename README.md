## 📖 About

This repository documents my **21-day preparation journey** for the **ICPC AlgoQueen 2026 Prelims**.

Instead of solving random questions, this challenge focuses on mastering the **most frequently tested algorithmic patterns** in competitive programming through carefully selected problems.

Every day includes:

- 🟢 One Easy problem
- 🟡 One Medium problem
- 🔴 One Hard / Contest-level problem

All solutions are implemented in **C++17**, accompanied by complexity analysis, key observations, and personal learnings.

## 🎯 Goals

- Solve **63 curated problems** in 21 days
- Strengthen Competitive Programming fundamentals
- Improve implementation speed in C++

- Maintain consistency through daily coding
